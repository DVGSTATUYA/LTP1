# LTP1

## Установка необходимого
Создайте виртуальное окружение и активируйте его:

```bash
python -m venv .venv
# Windows PowerShell:
. .venv/Scripts/Activate.ps1
# Linux/macOS:
source .venv/bin/activate
```
Установите необходимые библиотеки:
```
pip install -r requirements.txt
```


## Файл .env 
В папке app отредактируйте .env:
```env
DATABASE_URL=postgresql+psycopg://ИМЯПОЛЬЗОВАТЕЛЯ(стандартное: postgres):ПАРОЛЬ@localhost:5432/НАЗВАНИЕБД
```

### База данных 
Создайте базу данных в PostgreSQL и импортируйте таблицы через pgAdmin. 

## Запуск
Бэкенд FastAPI
```bash
cd app
uvicorn main:app --reload
```
Доступ: http://127.0.0.1:8000 
Документация: http://127.0.0.1:8000/docs

### Фронтенд Flask

```bash
cd app/frontend
python app.py
```

Интерфейс по ссылке: http://127.0.0.1:5000

Навигация:

*   Продукция
*   Цеха
*   Расчёт сырья
*   Добавить продукт


## ВНИМАНИЕ

*  Имена колонок должны совпадать с ожидаемыми в коде.
*  Имена модулей и файлов должны совпадать с импортами в коде.
*  Проверьте формат значений в таблицах (числа, проценты, даты) перед импортом.
*  Убедитесь, что таблицы заполнены и содержат обязательные поля.
*  Используйте Pydantic для валидации входных данных и возвращайте понятные сообщения об ошибках.
*  Включите логирование ошибок и трассировку запросов для быстрой диагностики.
